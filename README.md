# idl
>  Interface Definition Language

## Prerequisites
- [Go](https://go.dev/doc/install) 
- [buf](https://docs.buf.build/installation)

## Command

### Regenerate
```bash
$ make clean
$ make generate
```

### Lint
```bash
$ make lint
```

### Format
```bash
$ make format
```

### Manual
```bash
$ make help
```
