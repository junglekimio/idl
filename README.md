# idl
>  Interface Definition Language

## Requirements
- [Go](https://go.dev/doc/install) 
- [buf](https://docs.buf.build/installation)

## Command

### Regenerate
```sh
$ make clean
$ make generate
```

### Lint
```sh
$ make lint
```

### Format
```sh
$ make format
```

### Manual
```sh
$ make help
```
